NayiDisha AI 🚀
AI-Powered Voice-Led Workforce Screening Platform

NayiDisha AI is a modern AI-driven multilingual workforce screening platform designed to simplify and automate blue-collar and semi-skilled candidate hiring through voice-based interviews, AI evaluation, and fraud detection.

🌟 Features
🎙️ AI Voice Interview System
AI-generated interview questions
Voice-based candidate responses
Multilingual interview support
Real-time speech interaction


🧠 AI Evaluation Engine

Automated response evaluation


🛡️ Fraud Detection

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/debasri-pal/NayiDisha-AI.git
2️⃣ Frontend Setup
cd frontend
npm install
npm run dev

Frontend runs on:

http://localhost:5173
3️⃣ Backend Setup
cd backend
npm install
npm start

Backend runs on:

http://localhost:5000
🔐 Environment Variables

Create a .env file inside backend folder:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
